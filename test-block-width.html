Summary

Element Type Default Width 由内容撑开?
block (div, p, h1) 100% of parent ❌ NO
inline (span, a) Content width ✅ YES
inline-block Content width ✅ YES
In Flexbox/Grid Layout algorithm ⚠️ DEPENDS

Technical Details

Standard document flow (normal flow):

  • Block elements: width: auto = 100% of parent width
  • Inline elements: width: auto = content width

Why the confusion?

  • In Flexbox/Grid, width: auto behavior changes
  • Floated elements also behave differently (shrink-to-fit)
  • Absolutely positioned elements shrink to content

Key takeaway:

Don't say "div width is determined by content" - that's wrong in standard flow!
Say "div takes 100% parent width by default" ✅
